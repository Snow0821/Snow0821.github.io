# Curriculum Architecture

## Summary

The curriculum should support long-term growth and should separate concept learning
from past-problem judgment training.

## Current Shape

- Theory stage teaches core concepts.
- Each concept includes immediate practice problems.
- Past problems are classified by the theory or strategy needed.
- Past-problem solving is broken into explicit numbered steps.

## Next Actions

- Decide whether the first full track should be 6 months, 1 year, or both.
- Decide target student level before writing detailed weekly plans.
