# KOI Difficulty Calibration

## Summary

The user's concern is correct: elementary-division KOI problems can be quite hard.
Recent official KOI 2nd-round problem tables show elementary problems being shared
with middle-division and sometimes high-division problem positions.

## Public File

- `curriculum/04-koi-difficulty-calibration.md`

## Current Calibration

- Do not treat elementary problems as simple beginner exercises.
- Treat shared problems as bridge milestones between divisions.
- Use actual year/division/problem-number mappings when assigning stages.

## Examples

- 2024 `가로등`: 초2, 중1, 고1.
- 2024 `색깔 모으기`: 초3, 중2.
- 2023 `스케이트 연습`: 초2, 중1, 고1.
- 2023 `고기 파티`: 초4, 중3.
- 2022 `트리와 쿼리`: 초3, 중2, 고1.
