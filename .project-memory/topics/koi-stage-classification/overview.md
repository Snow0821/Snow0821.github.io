# KOI Stage Classification

## Summary

Based on recent KOI problem-sharing patterns, classify the olympiad curriculum as
a single 10-stage K ladder rather than separate elementary/middle/high ladders.

## Public File

- `curriculum/05-koi-stage-classification.md`

## Stages

- `K0`: entry implementation base
- `K1`: KOI early implementation/reading
- `K2`: elementary early stable
- `K3`: elementary middle / middle entry bridge
- `K4`: elementary late / middle early-middle
- `K5`: middle middle / high entry bridge
- `K6`: middle late / high early-middle
- `K7`: high middle
- `K8`: high late
- `K9`: high top / post-olympiad bridge

## Rationale

Recent KOI 2nd-round tables show shared problems such as elementary problem 2 being
middle problem 1 or high problem 1, and elementary late problems overlapping middle
middle positions.
