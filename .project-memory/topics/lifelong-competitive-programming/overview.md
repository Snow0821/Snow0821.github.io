# Lifelong Competitive Programming

## Summary

The project direction can expand beyond informatics olympiad preparation into a
lifelong competitive programming growth path, including Codeforces-style rating
progression and high-difficulty problem solving.

## Public File

- `curriculum/02-lifelong-competitive-programming-roadmap.md`

## Core Idea

The long-term curriculum should not only list algorithms. It should repeatedly train
modeling, candidate generation, proof, implementation, contest execution, and
upsolving.

## Stages

1. Programming and thinking basics
2. Olympiad fundamentals
3. Contest problem-solving stamina
4. Intermediate algorithms and proof
5. Advanced competitive programming
6. Research-style problem solving

## Next Actions

- Decide whether the main project should stay olympiad-centered or become a wider
  competitive programming curriculum.
- If widened, define separate tracks for 6 months, 1 year, and long-term master path.
