# Stage Folder Structure

## Summary

Created `curriculum/stages/` to organize K-stage goals, problem lists, and teaching
notes as separate small files.

Reorganized the public stages into continuous `K0`-`K9` folders. Ambiguous range
folders (`K4-K5`, `K5-K6`, `K6-K7`) are retained only under `_legacy-ranges/`.
Boundary problems should have one primary stage and may be referenced from
neighboring stages as bridge/challenge problems.
