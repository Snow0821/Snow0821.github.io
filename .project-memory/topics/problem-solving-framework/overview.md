# Problem Solving Framework

## Summary

Past problems should be solved through a repeatable diagnostic process, not only by
showing final solutions.

## Current Steps

1. Read the problem and extract constraints.
2. Generate candidate theories or strategies.
3. Select an approach and justify it.
4. Plan implementation.
5. Write code.
6. Verify and reflect.
