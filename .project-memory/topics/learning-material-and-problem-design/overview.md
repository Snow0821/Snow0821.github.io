# Learning Material And Problem Design

## Summary

The project now has a concrete operating design for how to build learning materials
and problem sets for each K0-K9 stage.

## Public File

- `curriculum/06-learning-material-and-problem-design.md`

## Core Structure

Each stage should include:
- concept notes
- judgment-question cards
- representative examples
- practice sets
- KOI bridge problems
- target problems
- upsolving and reflection records

## Problem Types

- Warm-up
- Concept Drill
- Variation
- KOI Bridge
- Target
