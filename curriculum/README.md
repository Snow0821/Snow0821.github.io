# Curriculum

This directory is the active curriculum area for the algorithm class site.

Current organization principle:

- Keep the curriculum as a visible tree.
- Put each new lesson, visualization, worksheet, or problem set under the
  closest matching branch.
- Use KOI problems as references and assignments, but keep official statements
  linked rather than copied.

Main map:

- `map.md`: human-readable curriculum tree.
- `problems/koi/`: official KOI problem, solution, test-data, and judge link
  indexes.

Planned branches:

- `foundations/`
- `data-structures/`
- `algorithms/`
- `problem-solving/`
- `stages/`
- `visual-notes/`
