# Algorithm Roadmap

## Summary

The curriculum now has a high-level algorithm roadmap from basic problem solving
through implementation, search, sorting, graph, DP, data structures, advanced graph,
strings, math, past-problem classification, and contest operations.

## Public File

- `curriculum/01-algorithm-roadmap.md`

## Current Order

1. Problem-solving basics
2. Brute force and simulation
3. Sorting and basic data processing
4. Two pointers and sliding window
5. Greedy
6. Graph traversal
7. Trees
8. Dynamic programming
9. Binary search
10. Data structures
11. Shortest paths and advanced graph
12. Strings
13. Math and combinatorics
14. Past-problem classification
15. Contest operations

## Next Actions

- Convert this roadmap into either a 6-month or 1-year weekly syllabus.
- Add concept-level practice problem templates after target level is decided.
