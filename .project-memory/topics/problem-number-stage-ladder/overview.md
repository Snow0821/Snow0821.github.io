# Problem Number Stage Ladder

## Summary

The curriculum now includes a granular stage ladder based on problem-number
milestones, such as elementary/middle/high division problem positions and later
Codeforces/ICPC-style milestones.

## Public File

- `curriculum/03-problem-number-stage-ladder.md`

## Structure

- `L0-L2`: entry-level programming and complexity basics
- `E1-E8`: elementary-division style milestones
- `M1-M8`: middle-division style milestones
- `H1-H8`: high-division style milestones through final olympiad-level problems
- `C1-C7`: post-olympiad competitive programming growth

## Notes

The problem-number labels are intended as adaptable difficulty milestones rather
than fixed claims about a specific year's official exam format.
